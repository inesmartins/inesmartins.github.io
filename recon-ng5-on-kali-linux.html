Recon-ng 5 on Kali Linux

Last update on October 2019.

1. Get the latest version of the Recong-ng

root@kali:/# apt-get update && apt-get install recon-ng

2. Start Recon-ng.

root@kali:/# recon-ng
...
[recon-ng v5.0.0, Tim Tomes (@lanmaster53)]
[*] No modules enabled/installed.

3. Refresh the marketplace index.

[recon-ng][default] > marketplace refresh
[*] Marketplace index refreshed.

4. Find a specific module.

[recon-ng][default] > marketplace search [module]

[recon-ng][default] > marketplace search hackertarget

Path Version Status Updated D K
recon/domains-hosts/hackertarget 1.0 not installed 2019-06-24

D = Has dependencies. See info for details.
K = Requires keys. See info for details.

5. Install a module

[recon-ng][default] > marketplace install [module_path]

[recon-ng][default] > marketplace install recon/domains-hosts/hackertarget

6. Start using a module

[recon-ng][default] > modules load [module_path]

7. List the selected module's options

List the module's name, author, version, description and options.
[recon-ng][default][hackertarget] > info

...
Options:
Name Current Value Required Description
SOURCE default yes source of input (see 'show info' for details)

8. Configure the module's options

[recon-ng][default][hackertarget] > info
options set SOURCE [host]
SOURCE => [host]

9. Run module

[recon-ng][default][hackertarget] > run

10. Return to "main menu".

[recon-ng][default][hackertarget] > back
[recon-ng][default] >
