OpenVAS CLI on Kali Linux

OpenVAS is an extremely popular open-source vulnerability assessment framework.

This tool is actually an open-source version of Nessus, which now has its own Community Edition.

Last update on November 2019.

1. Get the latest version of OpenVAS.

root@kali:/# apt update; apt install openvas

2. Configure OpenVAS.

Make sure you copy the password that's ouputted at the end of the configuration process.
root@kali:/# openvas-setup

3. A new browser tab should automatically display the https://localhost:9392/ address. You can login with admin/[the password you copied]

This tutorial doesn't really focuses on the web interface, but it's always good to know it exists.

4. Start using the CLI

root@kali:/# omp --help
