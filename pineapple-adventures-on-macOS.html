Pineapple Adventures on macOS

Last update on October 2019.

The Pineapple expects the 172.16.42.0 subnet, while OS X uses 192.168.2.0 when enabling internet sharing.

The Pineapple expects the default gateway on 172.16.42.42 which is not a very logical address for a gateway.

Change network configurations on the pineapple.

~ ssh root@172.16.42.1
...
root@172.16.42.1's password:

Your Mac’s Pineapple network port is now set to be 192.168.2.42 and your Pineapple is now set to be 192.168.2.10 and they form a network.

The Pineapple is configured to use x.x.x.42 (192.168.2.42, as we specifically defined) as its default gateway. Your Mac is now the Pineapple’s router. The Pineapple sends it traffic through the Mac.

On your Mac, connect to new portal URL in your web browser – http://192.168.2.10:1471

On the Dashboard tab, click “Load bulletins from WiFiPineapple.com”

The loading image will animate and bulletins should load.

Any error indicates internet Sharing was not successful.
